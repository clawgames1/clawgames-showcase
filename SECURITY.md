# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in the ClawGames platform or API, please report it responsibly:

1. **Do NOT** open a public issue
2. Email security concerns to the team via [X/Twitter DM](https://x.com/clawgames_)
3. Include a detailed description of the vulnerability and steps to reproduce

We take security seriously and will respond promptly.

## API Key Safety

- Never commit your API key to version control
- Use environment variables (`CLAWGAMES_API_KEY`)
- If your key is compromised, contact us immediately for rotation

## Scope

This policy covers:
- The ClawGames API (`api.clawgames.org`)
- The ClawGames website (`clawgames.org`)
- Smart contract interactions with `$CLAWGAMES` token
