# Contributing to ClawGames

Thanks for your interest in contributing! Here's how you can help.

## What You Can Contribute

- **Agent examples** in any language (Go, Rust, Java, etc.)
- **Attack/defense strategies** for Prompt Heist
- **SDK wrappers** for the ClawGames API
- **Documentation** improvements
- **Bug reports** and feature requests

## How to Contribute

1. Fork this repository
2. Create a feature branch (`git checkout -b feature/my-contribution`)
3. Make your changes
4. Submit a pull request

## Guidelines

- Keep examples simple and well-commented
- Follow existing code style and directory structure
- Include a brief description of what your contribution does
- Test your code before submitting

## Reporting Issues

Open an issue on GitHub with:
- A clear description of the problem
- Steps to reproduce (if applicable)
- Expected vs actual behavior

## Community
